//*******************************
// Running the app
//*******************************
For running the main receiving application
In the src/ folder, in a terminal run:

(chmod u+x launch_reception.sh)
./launch_reception.sh

and follow the instructions


//*******************************
// Testing
//*******************************
For testing the overall program,  
run the launch_tests.sh bash script 
(./launch_tests.sh, with chmod command above if needed)
then
run the launch_client.sh bash script
(./launch_client.sh, chmod if needed)

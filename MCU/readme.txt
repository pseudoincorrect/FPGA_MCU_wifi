.jxbrowser-data, .metadata., RemoteSystemsTempFiles are config files for code composer.
They are needed for to launch quickly the project in code composer.

The actual sources are in the Fpga_wifi_link folder.

To run the project properly the CC3200 SDK has to be installed since this project include linked files (present in the SDK outside of this project folder).
CC3200SDK: http://www.ti.com/tool/download/CC3200SDK

For more info, please read the user manual.
